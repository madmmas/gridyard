# Security Policy

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security reports.

Email the maintainer at the address on the GitHub profile for
[`madmmas`](https://github.com/madmmas), or open a private
[GitHub Security Advisory](https://github.com/madmmas/gridyard/security/advisories/new)
on this repository.

Include enough detail to reproduce the issue. We'll acknowledge receipt
and work on a fix before any public disclosure.
